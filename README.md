# rlcode

Reinforcement Learning Code Snippet

## Install

```bash
pip install git+https://github.com/KibaAmor/rlcode
```

## Devlopment

```bash
git clone https://github.com/KibaAmor/rlcode
cd rlcode
python -m venv .venv
source .venv/bin/activate
pip install -e '.[dev]'
```

## Test

```bash
python -m example.dqn_gym
```
